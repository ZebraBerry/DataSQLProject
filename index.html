My Data Project

By Stacey Smith

Initial Project Outline

This is my data project in which I intend to:

  • Find a raw data source/s
  • Store the data in a database
  • Use SQL to clean the data
  • Develop a data warehouse
  • Use the data to create visualisation

The aim of this project is to practise my skills that I learnt during uni in regards to data and learn some new skills, particularly:

  • Create a basic webpage using HTML & CSS
  • Find raw data for a database, maybe with an API or multiple data sources
  • Create a local database using SQL
  • Insert a lot of data into the database, maybe with Python
  • Data manipulation and cleaning
  • Create a datawarehouse
  • Design visualisations to present the data
